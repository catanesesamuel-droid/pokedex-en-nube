# PokéDex Cloud - Documentación Genérica del Frontend

## 🌟 Introducción
La interfaz de usuario de PokéDex Cloud es una aplicación web moderna, rápida y visualmente atractiva diseñada para entrenadores Pokémon. Permite a los usuarios explorar el vasto mundo de Pokémon, gestionar sus favoritos y armar equipos competitivos, todo con una experiencia de usuario fluida y temática.

## 🚀 Funcionalidades Principales

### 1. Autenticación y Perfil
- **Registro e Inicio de Sesión:** Sistema seguro para que cada entrenador tenga su propio espacio.
- **Gestión de Perfil:** Los usuarios pueden actualizar su nombre de usuario, correo y ver sus estadísticas (cuántos favoritos tienen, cuántos pokémons en su equipo, etc.).
- **Personalización:** Soporte para temas **Oscuro** y **Claro** para adaptarse a la preferencia del usuario.

### 2. Exploración de Pokémon
- **Listado Completo:** Visualización de pokémons con paginación.
- **Búsqueda Avanzada:** Filtro por nombre o número de Pokédex.
- **Filtros por Tipo:** Botones rápidos para filtrar pokémons por su tipo elemental (Fuego, Agua, Planta, etc.).
- **Detalle de Pokémon:** Información detallada que incluye tipos, estadísticas base, habilidades y versiones "Shiny".

### 3. Gestión de Colecciones
- **Favoritos:** Posibilidad de marcar pokémons como favoritos para tener acceso rápido a ellos en una sección dedicada.
- **Mi Equipo:** Herramienta para construir un equipo de hasta 6 pokémons, permitiendo definir el formato de combate (Singles, Doubles, VGC) y la visibilidad del equipo.

## 🎨 Diseño y Experiencia de Usuario
- **Estética "Retro-Moderna":** Uso de fuentes pixeladas (Press Start 2P) combinadas con una interfaz limpia y efectos visuales modernos.
- **Interfaz Responsiva:** La aplicación se adapta perfectamente a dispositivos móviles, tablets y ordenadores.
- **Feedback Visual:** Uso de "Toasts" (notificaciones emergentes) y loaders temáticos (Pokéball girando) para informar al usuario sobre el estado de sus acciones.

## 🛠️ Tecnologías Utilizadas
El frontend ha sido desarrollado con un enfoque de "Zero Dependencies" para maximizar la velocidad y compatibilidad:
- **HTML5:** Estructura de la aplicación.
- **CSS3 (Vanilla):** Estilos personalizados, variables CSS para temas y animaciones fluidas.
- **JavaScript (Vanilla):** Lógica de la aplicación, manejo del DOM y comunicación con la API.
